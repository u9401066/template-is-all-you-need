# 測試生成技能

## 描述
為指定的程式碼自動生成測試案例。

## 觸發條件
- 「生成測試」
- 「寫測試」
- 「test this」
- 「建立單元測試」

## 測試類型

### 單元測試
- 正常情況（happy path）
- 邊界條件
- 錯誤處理
- 空值/null 處理

### 整合測試（如適用）
- API 端點測試
- 資料庫操作測試

## 測試框架偏好

根據語言自動選擇：
| 語言 | 框架 |
|------|------|
| Python | pytest |
| JavaScript | Jest |
| TypeScript | Jest / Vitest |
| Go | testing |
| Rust | built-in |

## 輸出格式

```
## 測試案例

### 檔案：`test_xxx.py`

測試覆蓋：
- ✅ 正常流程
- ✅ 邊界條件
- ✅ 錯誤處理

[生成的測試程式碼]
```
